# 💼 React Developer Portfolio

A personal portfolio website built with **React.js** to showcase my projects and skills as a Front-End Developer.

## 🚀 Live Demo
👉 [View it here](https://react-portfolio-wine-pi.vercel.app/)

## 🛠️ Built With
- React.js
- Vite
- CSS Grid
- CSS-in-JS
- Custom Hooks (useInView)
- Deployed on Vercel

## ✨ Features
- Fully responsive design
- Smooth scroll animations
- Reusable components
- Clean, modern UI
- Live clock component

## 🖥️ Run Locally
git clone https://github.com/Salahhm25/react-portfolio.git
cd react-portfolio
npm install
npm run dev
